# Code of Conduct

StoryForge 社区致力于提供友好、专业、无骚扰的协作环境。

参与者应尊重不同经验、背景和观点；针对代码和事实提出具体反馈；接受善意纠正；避免侮辱、歧视、威胁、骚扰、公开他人隐私或发布秘密与凭据。

维护者可以编辑或删除不当内容，并对持续或严重违反行为的参与者采取临时或永久限制。需要报告行为问题时，请优先使用 GitHub 的私密报告渠道；不要在公开 Issue 中披露个人隐私。处理报告时，维护者应尽量保护报告者身份并以一致、相称的方式执行规则。

本规则适用于仓库、Issue、Pull Request、讨论以及代表 StoryForge 参与的其他公共空间。
